Consignes

Le but de l'exercice est de reproduire cette même page en utilisant le langage HTML et le langage CSS. Ces exercices n'auront pas d'impact sur l'obtention ou non du diplôme. En revanche, une note vous sera tout de même attribuée, afin d'évaluer votre niveau de compréhension des différentes notions abordées.
Le barème est donc le suivant :

  • 1 point pour la mise en forme du titre (police Roboto) et des consignes
  • 1 point pour chaque exercice
  • 1 points pour la cohérence / format de nommage des classes ou id (voir format Camel Case)
  • 1 point si toutes les classes, ids et commentaires sont en anglais
  • 2 points pour la lisibilité / simplicité du code

N'hésitez pas à utiliser toutes les ressources à votre disposition (Google, Mozilla Developer Network, etc.).
ATTENTION ! faites ces exercices par vous-même! ne vous faites pas aider par vos camarades via slack.
